# Exports

This folder contains exported artifacts in various formats for submission or sharing.

Supported export formats:
- **PDF** - Documents with proper formatting and citations
- **DOCX** - Microsoft Word compatible documents
- **PPTX** - PowerPoint presentations
- **CSV** - Spreadsheet data
- **HTML** - Web-viewable formats
- **MDX** - Markdown with extended syntax

All exports preserve:
- Academic citations and formatting
- Provenance metadata (optional)
- Watermark indicators for AI-generated content
- Original source attribution

Files in this folder are generated from artifacts in other folders (docs/, sheets/, decks/, etc.) and can be safely regenerated at any time.
