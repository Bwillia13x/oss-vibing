# Documents

This folder contains academic documents in MDX format with inline citation markers.

Each document includes:
- Structured content with headings
- Inline citation markers (e.g., [@smith2024])
- Embedded provenance metadata
- AI-generated sections marked as watermarked until accepted

## Example Structure

```mdx
---
title: "Research Paper Title"
author: "Student Name"
date: "2025-11-12"
style: "APA"
provenance:
  created: "2025-11-12T02:23:55.806Z"
  sources: []
---

# Introduction

This paper explores... [@smith2024]

# Background

...
```

Documents can be exported to PDF, DOCX, or other formats while preserving citations and formatting.
