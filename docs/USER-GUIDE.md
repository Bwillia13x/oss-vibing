## Getting Started

### What is Vibe University?

Vibe University is an academic workflow platform that combines:
- **AI-powered writing assistance** for all your academic work
- **Citation management** with real academic databases
- **Data analysis tools** for research and statistics
- **Presentation creation** for class projects
- **Export capabilities** to PDF, Word, PowerPoint, and Excel

**Key Features:**
- ‚úÖ **Academic Integrity by Design** - Every fact requires a source
- ‚úÖ **AI Watermarking** - Transparently mark AI-generated content
- ‚úÖ **Citation Verification** - Automatically check citation accuracy
- ‚úÖ **Unified Workspace** - All tools in one place
- ‚úÖ **Real-time Collaboration** (coming soon)

### System Requirements

- **Browser:** Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Internet Connection:** Required for AI features and citation lookup
- **Storage:** 100MB+ free space for documents and cache

3. Choose your sign-in method:

## Using AI Tools

Vibe University's AI Copilot helps you with every stage of academic work while maintaining academic integrity.

### The AI Copilot

The AI Copilot appears in the left sidebar and can:
- Answer questions about your work
- Find academic sources
- Generate outlines and drafts
- Verify citations
- Check grammar and style
- Suggest improvements

### AI Watermarking & Academic Integrity

**Important:** All AI-generated text is watermarked until you accept it.

**How it works:**
1. AI generates text with a light blue background
2. Review the generated text carefully
3. Click **Accept** to remove watermark and claim as your work
4. Or click **Modify** to edit before accepting
5. Or click **Reject** to remove and start over

**Why watermarking matters:**
- Ensures transparency about AI use
- Helps you track what you wrote vs. AI wrote
- Maintains academic integrity
- Prevents accidental plagiarism

**Best practices:**
- ‚úÖ Always review AI suggestions before accepting
- ‚úÖ Add your own thoughts and analysis
- ‚úÖ Modify AI text to match your voice
- ‚úÖ Never submit raw AI output without review
- ‚ùå Don't accept large blocks of text without reading
- ‚ùå Don't rely solely on AI for your arguments

## Managing Citations

### Adding Citations

**Method 1: Using AI Copilot**
```
Ask: "Add a citation for Smith 2023 about climate change"
```

**Method 2: Manual Entry**
1. Place cursor where citation should go
2. Click **Insert Citation** in toolbar
3. Enter DOI, URL, or search by title/author
4. Select citation and click **Insert**

**Method 3: Import from Reference Manager**
- Import from Zotero (coming soon)
- Import from Mendeley (coming soon)
- Import BibTeX file

### Citation Styles

Supported styles:
- **APA 7th Edition** (Psychology, Education, Social Sciences)
- **MLA 9th Edition** (Literature, Humanities)
- **Chicago 17th Edition** (History, Arts)
- **IEEE** (Engineering, Computer Science)
- **Harvard** (Business, Economics)
- **Vancouver** (Medicine, Life Sciences)

**To change citation style:**
1. Click **Settings** ‚Üí **Citation Style**
2. Select your preferred style
3. All citations update automatically

### Bibliography Generation

**Automatic bibliography:**
- Located at end of document
- Updates automatically when you add citations
- Formatted according to selected style
- Alphabetically sorted

**To customize bibliography:**
1. Click on bibliography section
2. Choose **Bibliography Settings**
3. Configure:
   - Hanging indent
   - Line spacing
   - Include/exclude URLs
   - Include/exclude DOIs

### Citation Verification

**Run verification check:**
1. Click **Tools** ‚Üí **Verify Citations**
2. Wait for analysis (usually <30 seconds)
3. Review results:
   - ‚úÖ Valid citations
   - ‚ö†Ô∏è Warnings (missing info, old sources)
   - ‚ùå Errors (broken DOIs, fabricated)
4. Fix issues one by one

**What gets checked:**
- DOI validity
- URL accessibility
- Author-title-year consistency
- Publication date accuracy
- Citation coverage (% of claims cited)

## Tips & Best Practices

### Writing Academic Papers

1. **Start with an outline**
   - Use the `outline_doc` tool
   - Break into manageable sections
   - Set word count goals for each section

2. **Research first, write second**
   - Use `find_sources` to gather literature
   - Read and annotate sources
   - Create citation library before writing

3. **Cite as you write**
   - Don't wait until the end to add citations
   - Insert citations immediately after claims
   - Use `verify_citations` periodically

4. **Edit in stages**
   - First draft: Get ideas down
   - Second draft: Improve structure
   - Third draft: Polish language
   - Final draft: Check citations and formatting

5. **Use AI as a research assistant, not a writer**
   - AI helps find sources and organize ideas
   - You provide the analysis and arguments
   - Always add your unique perspective

### Data Analysis

1. **Clean your data first**
   - Remove duplicates and errors
   - Handle missing values
   - Check for outliers

2. **Understand your statistics**
   - Don't just run tests blindly
   - Know what each test measures
   - Interpret results in context

3. **Visualize before analyzing**
   - Create charts to see patterns
   - Identify relationships visually
   - Catch obvious errors

4. **Document your methods**
   - Note which statistical tests you used
   - Record assumptions and limitations
   - Include sample sizes and p-values

### Collaboration (Coming Soon)

1. **Set clear roles**
   - Define who writes what
   - Assign research tasks
   - Agree on citation style

2. **Use version control**
   - Track who changed what
   - Revert to previous versions if needed
   - Review changes before accepting

3. **Communicate frequently**
   - Use comments for questions
   - Tag team members for feedback
   - Set deadlines and milestones

### Academic Integrity

1. **Always cite your sources**
   - Direct quotes need citations
   - Paraphrases need citations
   - Even common knowledge should be cited in academic work

2. **Never copy-paste without citations**
   - Use quotation marks for direct quotes
   - Paraphrase in your own words
   - Always attribute ideas to their authors

3. **Review AI-generated text carefully**
   - Don't accept without reading
   - Modify to match your voice
   - Add your own analysis and insights

4. **Run plagiarism check before submitting**
   - Use `detect_plagiarism` tool
   - Fix any flagged issues
   - Get originality report

5. **Keep a research log**
   - Track all sources you consult
   - Note when and how you used AI
   - Document your research process

## Troubleshooting

### Common Issues

#### Citations Not Formatting Correctly

**Problem:** Bibliography is not in the right style

**Solution:**
1. Go to **Settings** ‚Üí **Citation Style**
2. Select correct style (APA, MLA, Chicago, etc.)
3. Click **Refresh Bibliography**

**Problem:** DOI not found

**Solution:**
1. Verify DOI is correct (check publisher website)
2. Try searching by title and author instead
3. Manually enter citation details

#### Export Issues

**Problem:** PDF export is missing content

**Solution:**
1. Check if all sections are expanded (not collapsed)
2. Try **Preview** before exporting
3. Use **Advanced Export** options
4. Clear browser cache and try again

**Problem:** DOCX export loses formatting

**Solution:**
1. Use built-in styles rather than manual formatting
2. Check **Include Style Definitions** in export options
3. Open in Word and use **Update Styles** if needed

#### Performance Issues

**Problem:** App is slow or laggy

**Solution:**
1. Close unused documents
2. Clear browser cache: **Settings** ‚Üí **Clear Cache**
3. Disable browser extensions
4. Update to latest browser version

**Problem:** Large documents take long to load

**Solution:**
1. Enable **Progressive Loading**: **Settings** ‚Üí **Performance**
2. Split very long documents into chapters
3. Use **Offline Mode** for faster access

#### AI Copilot Issues

**Problem:** AI not responding

**Solution:**
1. Check internet connection
2. Refresh the page
3. Check if API key is set (institutional users)
4. Try rephrasing your question

**Problem:** Citations found are not relevant

**Solution:**
1. Be more specific in your search query
2. Add keywords like field, date range
3. Use advanced search filters
4. Try different search terms
