# Presentations

This folder contains presentation decks in JSON format.

Each deck includes:
- Slide outline with titles
- Slide content and layout
- Speaker notes for each slide
- Theme and styling information

## Example Structure

```json
{
  "title": "Research Presentation",
  "created": "2025-11-12T02:23:55.806Z",
  "theme": "academic",
  "slides": [
    {
      "id": 1,
      "title": "Introduction",
      "content": "Main points...",
      "speakerNotes": "Elaborate on..."
    }
  ],
  "provenance": {
    "sourceDoc": "docs/research-paper.mdx",
    "generatedAt": "2025-11-12T02:23:55.806Z"
  }
}
```

Decks can be exported to PPTX, PDF, or HTML formats.
