Search for academic sources using scholarly databases (Crossref, OpenAlex, Semantic Scholar) and return citations in CSL JSON format.

This tool helps students find credible academic sources for research papers. It:
- Searches multiple scholarly databases
- Returns metadata in standardized CSL JSON format
- Includes DOIs, URLs, and publication details
- Filters by year range and other criteria
- Stages results in the references/ folder for later use

All sources include provenance tracking with fetch timestamp for verification.
