Insert citations into a document with proper formatting in the specified citation style.

This tool adds citations to documents:
- Inserts citation markers at specified locations
- Formats citations according to style (APA, MLA, Chicago)
- Updates the document's citation list
- Maintains provenance tracking for all sources
- Marks insertions as AI-assisted until user accepts

Citations are inserted as inline markers (e.g., [@smith2024]) and can be formatted into a bibliography.
