Run academic integrity check on a document to verify citation coverage, quote accuracy, and identify watermarked content.

This tool ensures academic integrity by:
- Calculating what percentage of claims have citations
- Checking if quotes match their sources
- Identifying unresolved DOIs or URLs
- Finding watermarked AI-generated text that hasn't been accepted
- Providing one-click actions to fix issues

This is a critical tool for maintaining academic honesty and should be run before submitting any academic work.
