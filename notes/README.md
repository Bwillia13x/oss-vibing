# Notes and Study Materials

This folder contains study notes in MDX format with highlights, annotations, and flashcards.

Each note includes:
- Highlighted content from source materials
- Personal annotations and summaries
- Generated flashcards for spaced repetition
- Spaced repetition schedule metadata (SM-2 algorithm)

## Example Structure

```mdx
---
title: "Chapter 5 Notes - Biology"
source: "Biology Textbook, Chapter 5"
created: "2025-11-12T02:23:55.806Z"
flashcards: 15
nextReview: "2025-11-13T10:00:00.000Z"
---
```
